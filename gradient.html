How to open

그라디언트 패널 꺼내기(Ctrl+F9)

그라디언트 툴은 말그대로 대상을 단색 표현이 아닌 그라데이션 효과를 주고 싶을 때 사용되는 툴 입니다. 그라디언트를 사용하는 방법은 Window에서 Gradient를 누르거나 ctrl+f9을 누르면 그라디언트 패널을 볼 수 있습니다. 다른 방법으로는 왼쪽 툴박스에서 그라디언트 툴 아이콘을 더블클릭하여 사용할 수도 있습니다.

Set gradient color

Use Color

Gradient Slider를 더블클릭 한 후 컬러를 선택하여 각 항목별 조절을 하거나 원하는 컬러를 선택해서 그라데이션을 입힌다.

Use Swatches

원하는 컬러를 swatches에 새로 등록 한 후 그라디언트의 컬러 하단에 스와치를 선택하여 추가한 스와치 컬러를 이용한다.

Set gradient direction

Set Option

그라데이션의 Type을 선택해서 선형으로 할지 원형으로 할지 선택하거나 각도를 설정하여 원하는 방향으로 그라데이션을 만든다.

Use Drag

왼쪽 툴박스 에서 그라디언트툴을 선택한후 그라데이션을 적용한 매체위에 원하는 방향으로 드레그 하여 그라데이션 방향을 조절한다.

Learn more

Swatch Libraries Menu

Swatch libraries 에서 Gradients 를 선택하면 다양한 그라데이션을 볼 수 있다. 보기 옵션을 설정하여 썸네일도 설정가능 하다.

Add Gradient Color

컬러를 추가하고 싶은 위치에 마우스를 올려놓고 더블클릭을 하거나 슬라이더를 클릭한후 alt를 눌러 복사하여 색을 추가한다.
