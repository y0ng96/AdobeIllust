How to open

텍스트 탭 꺼내기


문자 툴을 선택하고 우측과 같은 탭이 활성화됩니다.
텍스트를 입력하기 위해서 대지를 클릭하면 라틴어로 된 자리 표시자 텍스트가 추가됩니다.

How to open

Path를 따라 텍스트 입력하기


예시를 위해, 원의 Path위에 텍스트를 입력하기 위해 Ellipse Tool을 활용하여 원을 그린 후, Type on a Path Tool을 선택합니다. 그 후, 원의 Path를 선택하면 Path를 따라 텍스트가 입력되는 것을 확인할 수 있습니다.

How to open

텍스트를 수직으로 입력하기


Text Tool 중, Vertical Type Tool을 선택합니다. 텍스트를 입력할 곳을 선택하면 우측과 같이 세로로 입력되는 것을 확인할 수 있습니다.

Text Pannel : 복잡하지 않아요

앞으로 자주쓰게 될 기능들 위주로 알아봅시다.

Text Pannel : 복잡하지 않아요

1. 폰트와 폰트의 굵기를 설정하는 부분입니다.

2. 폰트의 크기를 설정하는 부분입니다.

3. TT를 선택할 경우 선택한 텍스트를 대문자로, Tt를 선택할 경우 선택한 텍스트를 소문자로 바꿔줍니다.

Text Pannel : Ctrl + T

4. 텍스트의 행간(줄과 줄 사이)을 조절할 수 있습니다.

Text Pannel : 복잡하지 않아요

5. 텍스트의 자간(글자와 글자 사이)을 조절할 수 있습니다.
