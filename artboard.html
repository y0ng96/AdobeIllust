How to open

아트보드 툴 사용하기(Shift+O)

일러스트에서 아트보드 혹은 대지란 포토샵에서의 캔버스와 같은 의미입니다. 아트보드는 일러스트에서 작업영역을 뜻하며 하나 혹은 여러개의 아트보드를 사용하여 작업할 수 있습니다. 일러스트 사용시 아트보드가 부족한 경험이 있을 것 입니다. 그때는 새로운 아트보드를 생성을 해야합니다. 아트보드를 생성시 왼쪽 툴박스에서 아트보드 아이콘을 클릭하거나 단축키 shift+O 를 이용하여 만들 수 있습니다.

New Artboard

Set Option

Window 창의 아트보드 패널을 열어 아트보드를 추가해 주거나 아트보드 아이콘을 클릭한후 원하는 사이즈를 직접 입력한다.

Use Drag

아이콘을 클릭한후 드레그를 통해 아트보드를 생성한다. 또한 Alt+shift 를 하면 기존에 존재하던 아트보드와 같은 비율로 생성할 수 있다.

Move Artboard

Set Option

아트보드 툴을 누른후 아트보드 안쪽을 클릭하면 파란 바운딩 박스가 나온다 이때 원하는 위치로 아트보드를 옮길 수 있다.

Use Drag

Alt와 shift를 동시에 누르면서 아트보드를 드레그 하면 아트보드 안에 포함되어 있는 내용까지도 복사할 수 있다.

Artboard Guide Lines

Set Option

아트보드 패널에서 해당 아트보드를 선택한 후 오른쪽 아트보드 아이콘을 누르면 아트보드 옵션을 설정할 수 있다.

Set Option

아트보드 옵션에서 화면(display) 부분에 중심 표시(show center mark)와 십자선 표시(show cross hairs)를 선택하면 보조선이 나온다.

Rearrange All Artboards

Set Option

모든 대지 재정돈 아이콘을 통해 대지의 레이아웃, 순서, 열(columns), 간격(spacing) 을 조절할 수 있다.
