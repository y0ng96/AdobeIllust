How to open

도형툴 앱 꺼내기

도형툴을 활용하여 다양한 아트웍을 표현할 수 있습니다. 이러한 도형툴을 꺼낼 때에는 왼쪽 도형툴을 누르면 오른쪽과 같은 모습을 볼 수 있습니다. 또한 도형 툴 탭을 꺼내고 싶을 때에는 삼각형 아이콘을 누르면 왼쪽의 위와 같은 도형 툴 탭을 볼 수 있습니다. 이와 같은 방법은 펜 툴, 텍스트(타입) 툴 에서도 같은 방식으로 적용됩니다.

Rectangle Tool

Set Option

사각형 툴을 선택한 후 아무 곳이나 클릭하면 옵션 창을 볼 수 있습니다. 가로와 세로값을 입력하여 사각형을 만듭니다.

Use Drag

아트보드 위를 클릭, 드래그 해서 사각형을 그립니다. 이때 Alt와 Shift를 동시에 누르면 정사각형을 그릴 수 있습니다.

Rounded Rectangle

Set Option

라운드 사각형 툴을 선택한 후 아무 곳이나 클릭하면 옵션 창을 볼 수 있습니다. 각각의 값을 입력한 후 OK를 누릅니다.

Use Drag

Alt와 Shift를 동시에 누르면 라운드 정사각형을 그릴 수 있습니다. 또한 방향키를 이용하여 모서리 앵글을 조절 합니다.

Ellipse Tool

Set option

원형 툴을 선택한 후 아무 곳이나 클릭하면 옵션 창을 볼 수 있습니다. 각각의 값을 입력한 후 OK를 누릅니다.

Use Drag

아트보드 위를 클릭, 드래그 해서 원을 그립니다. 이때 Alt와 Shift를 동시에 누르면 정원을 그릴 수 있습니다.

Polygon Tool

Set option

다각형 툴을 선택한 후 아무 곳이나 클릭하면 옵션 창을 볼 수 있습니다. 각각의 값을 입력한 후 OK를 누릅니다.

Use Drag

클릭과 드래그를 하면서 마우스를 떼지 않은 채로 방향키를 이용하여 변 수를 조정할 수 있습니다.

Star Tool

Set option

별 툴을 선택한 후 아무 곳이나 클릭하면 옵션 창을 볼 수 있습니다. 각각의 값을 입력한 후 OK를 누릅니다.

Use Drag

클릭과 드래그를 하면서 마우스를 떼지 않은 채로 방향키를 이용하여 꼭짓점 수를 조정할 수 있습니다.

Flare Tool

Set option

플레어 툴을 선택한 후 아무 곳이나 클릭하면 옵션 창을 볼 수 있습니다. 각각의 값을 입력한 후 OK를 누릅니다.

Set option

순서대로 각각의 옵션들은 1. 중심광원 2. 후광(광륜) 3.광선 4. 링 을 나타냅니다.
